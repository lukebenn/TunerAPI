◆ I2Cinit()

int  mcuClock 
)

Initalizes I2C module with default pins.

Initalizes I2C module by:

  • setting the clock to on
  • adjust the switch matrix
  • fully reseting I2C module
  • setting baud rate to default 100Khz, 12Mhz

I2C_MasterSetBaudRate() command must be called again to adjust baude rate or declare new FRO speed.

Used to force program to wait until I2C bus is in correct state.

Waits for I2C mater pending, then compares the master state parameter. If compare fails, enter a while(1) loop. If compare passes, return.

Parameters
ptr_LPC_I2CA pointer to an I2C instance
stateOne of the 3-bit Master function state codes of the I2C - Refer to LPC802 manual table 187
